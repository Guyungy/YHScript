# 油猴脚本

收录了一些自用**TamperMonkey**[油猴脚本](https://greasyfork.org/zh-CN)
如果觉得不错，点个`Star`吧 😃  


## Install

首先安装[TamperMonkey](https://www.tampermonkey.net/)或者[暴力猴](https://violentmonkey.github.io/)拓展，也就是俗称的油猴拓展，版本库中有如下脚本，可以直接点击安装按钮进行安装。
